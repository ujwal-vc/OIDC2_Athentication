About This App

This is a sample authentication app using Auth0. It demonstrates how to integrate secure login and logout features into a web app with minimal code and maximum reliability.

The application supports authenticated sessions, user profile retrieval, and displays session details once logged in.

Built with ❤️ using HTML, Python (Flask or similar), and Auth0.
